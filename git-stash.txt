
правильный сташ

    1. Добавляем в индекс
        git add .
    2 git stash

Все изменения складываются в стешь лист его можно посмотреть

    git stash list

Накатить из stash можно выполнив команду, его можно
    git stash apply // по умолчанию накатится последний стешь
    git stash apply{1} явно указать какой стешь хотим накотить

сбросить стешь
    git stash drop // Он тоже как я понял удаляет по одному
