1. установить гиту кто ты такой    git config --global user.name "gordondalos"    git config --global user.emai "gordondalos@gmail.com"2. посмотреть текущие настройки        git config --list3. посмотреть где этот конфиг        git config --list --show-origin4. создать проект        git init5. Добавить все что есть, или конкретный файл        git add .        git add git.md6. Создаем комит    git commit -m 'Первый комит'7. Задать редактор нано по умолчанию        git config --global core.editor nano        git config --global core.editor notepad7а настроить чтобы понимал кирилицу (не работает как ожидалось)        git config --local core.quotepath false        git config --local core.commitencoding  utf8        git config --local i18n.logoutputencoding utf8        git config --local core.pager "iconv.exe -f cp1251 -t utf-8 | less"8. создать коммит без открывания нотепада        git commit -m 'мой комментарий'9. Скачать репозиторий    git clone ТУТАДРЕССВНЕШНЕГОРЕПО9. Смотрим историю    gitk10 Запускаем гуевину    git gui11 Посмотреть статус проекта (сокращенная версия)    git status    git status -s12. Посмотреть разницу между изменениями    git diff  // а так можно посмотреть только то что изменилось    git diff --cashed // это посмотреть что будет добавлено в комит13 Сразу добавить изменения и коментарий    git commit -am "Сразу все добавил и закоментировал"14. Откат незафиксированных измененийчтобы удалить изменения в файле можно просто написать   git checkout -- text.txt // это только в том файле который указать
